# Set

### 这是什么？
`set` 是 C++ STL 中的一种关联容器，它维护了一个**元素互不相同**且**自动排序**的集合。 在底层实现上，它通常是一棵**红黑树（Red-Black Tree）**，这是一种自平衡的二叉搜索树（BST）。

- 核心特性：**元素唯一（自动去重）+ 自动排序**。
- 核心优势：
  - 无需手动去重、排序，适合需要「有序无重复集合」的场景（如统计唯一元素、二分查找）；
  - 查找 / 插入 / 删除效率高（平均 O(logn)）。

### 常用方法举例：

```cpp
#include <set>
using namespace std;

// 1. 定义：创建一个存放整数的 set，默认从小到大排序
set<int> s;

// 2. 插入：插入元素，自动去重并排序
s.insert(10);
s.insert(5);
s.insert(10); // 重复插入无效，集合内仍为 {5, 10}

// 3. 删除
s.erase(5); // 集合变为 {10}

// 4. 查找是否存在
if (s.count(10)) { ... } // 返回 1 表示存在，0 表示不存在

// 5. 核心功能：二分查找（Lower Bound）
// lower_bound(x) 返回指向第一个 >= x 的元素的迭代器
// upper_bound(x) 则返回第一个 > x 的元素的迭代器
auto it = s.lower_bound(8); 

// auto的用法是根据赋值内容自动选择合适的数据类型
if (it != s.end()) {
    int val = *it; // 获取值
} else {
    // 没找到，说明所有元素都比 8 小
}

// 6. 遍历
for (auto x : s) { ... }
```
