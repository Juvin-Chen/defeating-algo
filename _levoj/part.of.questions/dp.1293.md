# P1293 放苹果

#### 题目描述

> 小明刚放学回家，老妈就吩咐他：马上有客人要来，赶快把苹果洗了放到盘子里去。小明要把 m 个同样的苹果放到 n 个同样的盘子里，允许有的盘子空着不放，有多少种不同的 分法。5,1,1 与 1,5,1 是同一种放法。

#### 输入描述

> 多组输入，输入两个数 m 和 n 代表苹果数和盘子数，m 和 n 均不超过 20

#### 输出描述

> 对于每一组数，有多少种不同的放法

#### 样例输入

```
7 3 
```

#### 样例输出

```
8
```

# 放苹果 - 递归解法

## 题目分析

将 m 个相同苹果放到 n 个相同盘子里（允许空盘），不考虑顺序（如 5,1,1 和 1,5,1 算同一种），求不同分法数。

## 递归解法（核心：状态定义与转移）

### 1. 状态定义

定义递归函数 `f(m, n)`：表示将 `m` 个苹果放到 `n` 个盘子里的**不同分法数**。

### 2. 递归终止条件

1. **无苹果可放**（m=0）：只有 1 种分法（所有盘子空），返回 1；
2. **无盘子可放**（n=0）：无法放置，返回 0；
3. **盘子数 > 苹果数**（n > m）：空盘子不影响分法，等价于 `f(m, m)`（去掉多余空盘子）。

### 3. 状态转移（核心逻辑）

分两种情况，覆盖所有可能的放法：

- **情况 1**：至少有 1 个盘子空着 → 等价于 `f(m, n-1)`（少用 1 个盘子）；
- **情况 2**：所有盘子都有苹果 → 先给每个盘子放 1 个苹果（共放 n 个），剩余 `m-n` 个苹果再分配 → 等价于 `f(m-n, n)`。

总分法数 = 情况 1 + 情况 2，即：

```
f(m, n) = f(m, n-1) + f(m-n, n)
```
