# 洛谷动态规划 (DP) 路线图

## 一、 线性 DP：培养状态定义的直觉

这是所有 DP 的基石。重点练习如何把大问题拆解成小问题，以及一维/二维状态数组的定义。

- **基础递推（找规律找感觉）：**
  - `P1216` [USACO1.5] 数字三角形 (DP 绝对入门，体会自底向上的推导)
  - `P1115` 最大子段和 (极简的一维 DP 模型)
  - `P1044` [NOIP2003 普及组] 栈 (卡特兰数 / 简单基础 DP)
- **子序列问题（经典老番）：**
  - `P1020` [NOIP1999 普及组] 导弹拦截 (最长不下降子序列 LIS，重点掌握 $O(N \log N)$ 优化)
  - `P1091` [NOIP2004 提高组] 合唱队形 (LIS 的经典变种应用)
  - `P1439` 【模板】最长公共子序列 (LCS 模板题)
- **字符串与二维线性：**
  - `P2758` 编辑距离 (极其经典的二维字符串 DP)
  - `P1283` 乘积最大 (结合高精度运算的 DP)

## 二、 背包 DP：蓝桥杯的半壁江山

背包问题不仅是必考点，更是很多复杂算法的底层逻辑。必须刷到闭着眼睛都能写出状态转移方程和一维空间优化的程度。

- **01背包及其变种（基础必秒）：**
  - `P1048` [NOIP2005 普及组] 采药 (标准的 01 背包模板)
  - `P1049` [NOIP2001 普及组] 装箱问题 (01 背包变种，求最小剩余空间)
  - `P1164` 小A点菜 (求方案数的 01 背包变种)
- **完全与多重背包（进阶拓展）：**
  - `P1616` 疯狂的采药 (完全背包模板，重点体会数据类型防爆 `long long`)
  - `P1833` 樱花 (混合背包：01 + 完全 + 多重，多重背包的二进制拆分优化是关键)
- **分组与依赖背包（拉开差距的考点）：**
  - `P1757` 通天之分组背包 (分组背包模板题)
  - `P1064` [NOIP2006 提高组] 金明的预算方案 (有依赖的背包，逻辑分类要非常清晰)

## 三、 区间 DP：破解环形与合并难题

特征是“大区间由小区间合并而来”。最外层循环枚举区间长度，内层循环枚举左端点，再枚举分割点。

- **经典模板与断环成链：**
  - `P1880` [NOI1995] 石子合并 (祖师爷级别的题，必刷，掌握环形处理技巧)
  - `P1063` [NOIP2006 提高组] 能量项链 (石子合并的具象化变种)
- **进阶场景模型：**
  - `P3146` [TJOI2015] 248 G (类似 2048 游戏的区间合并逻辑)
  - `P1220` 关路灯 (加上了时间/消耗成本的区间 DP，极其考验状态定义)
  - `P3205` [HNOI2010] 合唱队 (从两侧向中间推导的区间 DP)

## 四、 树形 DP：冲刺高分与省一的利器

将 DP 逻辑搬到了树状结构上，通常结合 DFS 递归实现（自底向上回溯更新状态）。

- **基础树形状态转移：**
  - `P1352` 没有上司的舞会 (树形 DP 第一题，理解 `dp[u][0]` 和 `dp[u][1]` 的含义)
  - `P1131` [ZJOI2007] 时态同步 (理解树上路径长度调整逻辑)
- **树上背包（高频且硬核）：**
  - `P2014` [CTSC1997] 选课 (将有依赖的背包问题转化为树形 DP)
  - `P2015` 二叉苹果树 (经典的树上背包入门)
  - `P3177` [HAOI2015] 树上染色 (极其考验时间复杂度优化的树上背包进阶题)

## 五、 状态压缩 DP & 数位 DP：降维打击

这类题目属于比赛后半程的防 AK（All Kill）题，掌握后能在难题中硬抠出可观的分数。

- **状态压缩 DP（利用二进制位运算表示集合状态）：**
  - `P1879` [USACO06NOV] Corn Fields G (棋盘类状压入门，熟练掌握位运算 `&`, `|`, `<<`)
  - `P1896` [SCOI2005] 互不侵犯 (经典状压，结合了行与行之间的状态约束)
  - `P2704` [NOI2001] 炮兵阵地 (需要保存前两行状态的状压 DP，挑战代码能力)
- **数位 DP（蓝桥杯特爱出的题型，统计特定区间内满足条件的数字个数）：**
  - `P2602` [ZJOI2010] 数字计数 (数位 DP 模板，学会用记忆化搜索实现)
